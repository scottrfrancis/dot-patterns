# overlays/ey — EY engagement patterns

Client/engagement-specific patterns and practices. **Never published to the public mirror.**
Shareable only to EY teammates via a controlled lateral channel (a scrubbed package or an
internal repo), never back to the public LAN mirror. `visibility: ey`.

Anything here is presumed to contain (or risk) client/employer identifiers — the publish
pipeline's `scrublist.public` must exclude this whole tree, and the `ey` scrub profile
governs lateral sharing.
